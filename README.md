# Introducción a Algoritmos y Estructuras de Datos

El objetivo de este repo es proporcionar una introducción clara y concisa a los conceptos fundamentales de algoritmos y estructuras de datos. Aquí encontrarás explicaciones teóricas, ejemplos prácticos y ejercicios para ayudarte a comprender mejor estos temas esenciales en la informática.

Toda la clase está dentro del archivo `E166 - Algoritmos y Estructuras de Datos en entrevistas técnicas.qmd`. Para poder renderearlo a html u otro formato, necesitas usar Quarto. Yo recomiendo usar Positron para leerlo, ya que tiene Quarto instalado por defecto. Pero también puedes instalar Quarto en tu computadora siguiendo las instrucciones en https://quarto.org/docs/get-started/.
